# 📚 Data & AI Learning Resources

HELLOO, Welcome to my open-source repository for **Data and Artificial Intelligence learning resources**.

This repository is created to share and organize useful resources related to **Machine Learning, Deep Learning, Big Data, Data Analysis, and Data Engineering**.

The goal is to help students, beginners, and data enthusiasts find helpful materials to learn, practice, and grow in the data field.

---

## 🎯 Main Idea

This repository is a curated collection of learning resources such as:

- 📚 Books
- 🎓 Courses
- 🛠️ Tools
- 🧾 Cheat sheets
- 🗺️ Roadmaps
- 💡 Project ideas
- 🔗 Useful links

It is designed to make learning easier by organizing resources in one place.

---

## 🧩 Topics Covered

### 📊 Data Analysis
Resources about data cleaning, exploratory data analysis, statistics, SQL, Python, dashboards, and data visualization.

### 🤖 Machine Learning
Resources about supervised learning, unsupervised learning, classification, regression, feature engineering, and model evaluation.

### 🧠 Deep Learning
Resources about neural networks, computer vision, natural language processing, TensorFlow, Keras, and PyTorch.

### ⚡ Big Data
Resources about Hadoop, HDFS, Hive, HBase, Apache Spark, PySpark, and distributed data processing.

### 🏗️ Data Engineering
Resources about data pipelines, ETL, ELT, data warehouses, data lakes, orchestration, and data quality.

### ☁️ Cloud Data Platforms
Resources about Azure, AWS, Google Cloud, Databricks, cloud storage, and cloud-based data pipelines.
